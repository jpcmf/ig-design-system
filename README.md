# Lab Design System

### install

```
npm install
```

### run app

```
npm run dev
```

### configure tailwind

```
npx tailwind init -p
```

### configure storybook

```
npx sb init --builder @storybook/builder-vite --use-npm
```

### configure clsx

```
npm install --save clsx
```

### configure @radix-react-slot

```
npm install @radix-ui/react-slot
```
